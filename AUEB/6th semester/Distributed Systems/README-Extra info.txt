How to run:

Για τον Broker:
 1) Άμα χρησιμοποιείτε κάποιο IDE αντί για cmd, θέστε ως το working directory τον src ή bin φάκελο.
 2) Να ανοίξειτε τους brokers (όσους θέλετε) ΠΡΙΝ τρέξετε την εφαρμογή.
 3) Να διαγράφετε ό,τι έχει το configuration file μέσα του πριν τρέξετε τους brokers.

Για το Android:
 1) Χρειάζεται redirect από telnet μέσω του terminal άμα η εφαρμογή τρέξει σε emulator. Δείτε το configuration file
    των brokers για να μάθετε ποια είναι τα ports των AppNodes που συμβολίζονται με A
    π.χ. redir add tcp:10750:10750
 2) Για server IP θέστε το IP address που εκτυπώθηκε στους brokers.
 3) Άμα βάλετε την εφαρμογή σε κινητό αξίζει να γνωρίζετε πως μπορεί ορισμένα xml αρχεία να φαίνονται περίεργα
    και να βγαίνουν εκτός οθόνης, ή γενικά να μην τρέχει καν ανάλογα το κινητό.

Small Extras:

1) Όλα όσα αναφέρθηκαν στον README του πρώτου παραδοτέου (εκτός από το τρίτο)
2) Στη χρήση EditText ο χρήστης μπορεί να χρησιμοποιήσει μόνο συγκεκριμένους χαρακτήρες που επιτρέπονται ανάλογα
   με την περίσταση π.χ. δεν μπορεί να χρησιμοποιήσει κεφαλαία γράμματα όταν ορίζει hashtags. Για την λειτουργία
   αυτή έγινε και η χρήση φίλτρων σε ένα activity.
3) Ο χρήστης δεν περιμένει για τίποτα, το πρόγραμμα τρέχει στο background και ο χρήστης μπορεί να διαβάζει
   κατανεμημένα συστήματα ενώ λαμβάνει βίντεο από topics που έχει κάνει subscribe.
4) Για κάθε είσοδο χρήστη σε EditText γίνεται κάθε έλεγχος στο αν η είσοδος είναι έγκυρη ή όχι και εμφανίζεται
   κατάλληλο μήνυμα αν κάτι πάει λάθος.
5) Ίσως ένα από τα πιο χρήσιμα features, ο χρήστης μπορεί να αλλάξει την φωτογραφία προφίλ του άμα πατήσει στο
   κουμπί πάνω δεξιά στο αρχικό μενού κι να διαλέξει μία από τις πανέμορφες 8 φωτογραφίες που προσφέρει η εφαρμογή.
6) Ένα υπέροχο UI που κάνει κάθε χρήστη να νιώθει σαν το σπίτι του όσο χρησιμοποιεί την εφαρμογή.

Notes:

1) Και στο configuration file αλλά και στο log των brokers, οι Brokers παρουσιάζονται με "B" και κάποιον αριθμό
ενώ οι AppNodes με "A" και το όνομα του αντίστοιχου χρήστη.

2) Στο configuration file η πρώτη στήλη είναι το ID ενώ η δεύτερη στήλη το port του κάθε instance.

3) Ο κώδικας των Brokers έχει αλλάξει λίγο από το πρώτο παραδοτέο. Η τελευταία και σωστή έκδοση των Brokers
   που τρέχει σε συνδιασμό με το Android βρίσκεται φυσικά σε αυτόν τον φάκελο. Η βασική αλλαγή είναι πως τώρα
   οι Brokers αποθηκεύουν στην λίστα των AppNode τους και το port αλλά και την IP κάθε AppNode, κάτι το
   οποίο δεν γινόταν στο πρώτο παραδοτέο γιατί όλα έτρεχαν στον ίδιο υπολογιστή κάτω από την ίδια IP.
   Επίσης οι Brokers έχουν και μια καινούρια λειτουργία "initialize me" που προσφέρουν στους AppNodes μιας και
   στο παραδοτέο αυτό οι AppNodes δεν έχουν πρόσβαση στο configuration file των Brokers.

4) Και το AppNode Thread αλλά και το Broker Thread χρησιμοποιούν strings μεγέθους 2-3 γραμμάτων που τους δίνουν οι
AppNodes / Brokers που συνδέονται μαζί τους και βάση αυτών μαθαίνουν τι λειτουργία πρέπει να κάνουν.

5) Μαζί με το πρώτο παραδοτέο
- 25 μέρες δουλειάς
- 1800+ γραμμές κώδικα.
- πολύς κόπος
